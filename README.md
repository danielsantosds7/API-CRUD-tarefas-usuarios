# API-CRUD-tarefas-usuarios

CRUD projetado em Web API com conexão com banco de dados SQL Server.
# 

<b>Funcionalidades:</b>

<b>Usuário</b>
 - Cadastrar usuário
 - Alterar usuário
 - Listar todos usuários
 - Deletar usuário

<b>Tarefas</b>
 - Cadastrar tarefa
 - Alterar tarefa
 - Listar todas tarefas
 - Deletar tarefa
